# LiveKit Voice Agent Frontend

React frontend for connecting to the LiveKit voice agent.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Update `TOKEN_SERVER_URL` in `src/App.tsx` if your backend runs on a different port.

3. Start the development server:
```bash
npm run dev
```

4. Open http://localhost:5173 in your browser.

## Features

- Connect to LiveKit voice agent
- Audio visualizer showing agent state
- Microphone controls
- Real-time agent state display
